What is this?

UDP logserver. Purpose is to collect excessive logging and save it to file without latency.

Why this?

An UDP logger removes the latency from connecting to a server or writing to a filehandle. We then stream the file into a tool like Splunk or Mixpanel. That way we can store both formats for future tools.

How does this work?

You create an UDP package inside you code and send it to the server without waiting for a reply.

PHP howto

<?php
    $server_ip   = "127.0.0.1";
    $server_port = 8080;
    $message     = "Hello World";
    print "Sending req to IP $server_ip, port $server_port\n";

    if ($socket = socket_create(AF_INET, SOCK_DGRAM, SOL_UDP))
        socket_sendto($socket, $message, strlen($message), 0, $server_ip, $server_port);
    else
        print("cannot create socket\n");
?>
